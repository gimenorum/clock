# 天文時計 · Horologium Astronomicum

実際の空の動きに合わせて時を刻む、天文時計の Web アプリです。いまの時刻に合わせて、太陽や月、惑星の位置、月の満ち欠けが文字盤に映ります。インストール不要で、ブラウザですぐに楽しめます。

## 2 つの文字盤

上部のボタンで 2 種類の表示を切り替えられます。

- **天文時計** — 金と紺色の文字盤。金色の太陽と銀色の月が空をめぐり、月の満ち欠けや、昼と夜の境目がわかります。
- **太陽系** — 星空のようにきらめく文字盤に、太陽を囲んで惑星（水星から海王星まで）が並びます。中央の太陽が「分」、月の形をした針が「秒」、短い針が「時」を指します。

## できること

- いまの太陽・月・惑星のおおよその位置を表示
- 月の満ち欠けを今夜の姿で表示
- 日付を前後に動かして、過去や未来の空を眺める
- 「早送り」で時間を速く進める

## 操作ガイド

### 表示を切り替える
- 上の「天文時計／太陽系」で、時計の種類を選びます。
- その下の「アナログ／デジアナ／デジタル」で見た目を選びます（順に、文字盤のみ／文字盤＋数字／数字のみ）。
- 文字盤を左右にスワイプしても切り替えられます。

### 全画面で使う（置き時計として）
- 文字盤（デジタル表示のときは数字）を**タップ**すると全画面になります。
- 元に戻すときは、**2本指でタップ**するか、**Esc** キーを押します。素早く2回タップすると、戻し方のヒントが出ます。
- iPhone・iPad では「ホーム画面に追加」しておくと、よりすっきりした全画面で起動できます。

### 時間を動かす
- 「天文時計／太陽系」の右にある「**早送りをオン**」にチェックを入れると、操作エリアが開きます。
- **日付スライダー**を動かすと、過去・未来の空（最大 ±約1年）を眺められます。
- 「**早送り**」で時間が速く進み、太陽・月・惑星の動きを早回しで見られます（「停止」で止まります）。
- 「**今に戻す**」で現在の日時に戻ります。チェックを外すと操作エリアは閉じ、現在時刻に戻ります。

## ちょっとした注意

表示される位置や満ち欠けは、楽しんだり眺めたりするための「おおよその計算」です。日食や惑星の並びを正確に予測するものではありません。

本アプリは個人が趣味で制作した非公式のプロジェクトで、特定の時計メーカーや製品、ブランドとは関係ありません。
